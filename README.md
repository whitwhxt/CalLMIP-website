# CalLMIP Website

Official website for the Canadian Land Model Intercomparison Project (CalLMIP).

## Structure

This website includes the following pages:

- **About** - Project overview and information
- **Protocol and Data** - Protocol and data guidelines
- **Phase 1** - Phase 1 documentation and resources
  - Getting Started
  - Protocol
  - Data
  - Co-authorship
- **Phase 2** - Phase 2 documentation
- **Participants** - List of participating institutions
- **Events** - Project events and workshops
  - CalLMIP 1st Phase Planning Workshop
- **Resources** - Additional resources and links

## Setup

This is a static website that can be hosted on GitHub Pages.
